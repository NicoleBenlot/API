# Product Inventory - Blazor + PHP RESTful API

## Requirements
- .NET 9 SDK
- A browser

## How to run
1. Clone the repo
2. Open terminal in the project folder
3. Run: dotnet run
4. Go to: http://localhost:5157/products (port could be different on other device)

## How it works
- Frontend: Blazor Server (C#)
- Backend: PHP REST API hosted on HelioHost
- Database: MySQL (hosted on HelioHost)
